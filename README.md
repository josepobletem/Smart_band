# Smartband MVP

## Funcionalidad

- Sugerencias de ejercicios en tiempo real según FC.
- Variables: Estatura, grasa corporal.
- Simulación o conexión real por serial (ESP32).
- API REST con FastAPI.
- Historial en JSON y SQLite.
- GUI con Tkinter.
- Docker y entorno virtual.

## Uso

```bash
make install
make run
```
